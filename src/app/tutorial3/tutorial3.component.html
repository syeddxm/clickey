EARDU SCANNING TUTORIAL

First, you will practice using the scanning method on the EARDU keyboard

1. Press either Shift key to begin

2. The keyboard will begin scanning downwards

3. When highlighted row contains the desired key, press either Shift key again

4. The keyboard will begin scanning from left to right

5. When desired key is highlighted, press either Shift key again

6. Press either Shift key to begin again

There is no way to backspace, just continue to writing the next key
